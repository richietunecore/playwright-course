# 📘 Module 1 – Introduction to Playwright

---

## ✅ What is Playwright?

Playwright is a modern **end-to-end test automation framework** developed by Microsoft.

It allows you to automate:

- 🌐 Web applications
- 📱 Mobile web
- 🔌 APIs
- 🧪 Cross-browser testing

---

## ✅ Why Playwright is Powerful

Playwright supports:

- ✅ Chromium
- ✅ Firefox
- ✅ WebKit (Safari)

With a **single API**.

It also includes:

- Auto-waiting
- Built-in assertions
- Network interception
- Parallel execution
- Headless & headed mode
- Trace viewer & debugging tools

---

## ✅ Playwright vs Selenium

| Feature | Playwright | Selenium |
|----------|------------|------------|
| Auto-wait | ✅ Built-in | ❌ Manual waits often needed |
| Speed | ⚡ Faster | Slower |
| Modern Architecture | ✅ Yes | Older architecture |
| Multiple Tabs | ✅ Easy | More complex |
| Network Interception | ✅ Built-in | Limited |

---

## ✅ How Playwright Works

Basic flow:

1. Launch browser
2. Open page
3. Locate element
4. Perform action
5. Assert result
6. Close browser
